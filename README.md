# Technologie Chmurowe — Zadanie 1 #

## Autor: Ireneusz Witek ##

## Opis zadania: ##
Aplikacja jest prostą stroną internetową uruchamianą w kontenerze Docker, która umożliwia sprawdzenie podstawowych informacji o aktualnej pogodzie dla wybranego miasta. Dane pogodowe pobierane są z serwisu wttr.in i prezentowane w przejrzystym interfejsie WWW.
